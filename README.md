### Description

Asustor app to install Open Speedtest and run it in a GUI to test your LAN speed.

### Download the package

1. Download the latest version .apk file from https://github.com/007revad/Asustor_Open_Speedtest/releases and save it to your Asustor.
2. In App Central click on Management then Manual Install.
3. Browse to where you downloaded the .apk file.
4. Select the .apk file and click Upload.
